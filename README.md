# 📹 Apna Video Call (MERN + WebRTC)

A real-time video calling web application built using **MERN Stack + WebRTC + Socket.IO**.  
It enables peer-to-peer video communication with room-based joining system.

## ✨ Features

- 🔴 Real-time video & audio calling
- 👥 Room-based meeting system
- 🔗 Peer-to-peer WebRTC connection
- 💬 Socket.IO signaling for connection setup
- 📡 Low-latency communication
- 📱 Responsive UI for all devices

---

## 🧠 How It Works

1. User creates or joins a room
2. Socket.IO handles signaling between users
3. WebRTC establishes direct peer-to-peer connection
4. Video/audio stream starts in real time
